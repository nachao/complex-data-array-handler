# nArray

### 获取数据（主要功能）
```javascript
[].$get();
```

##### $get 参数说明：
```javascript
@ {string} 获取条件
@ {boolean} 是否必须满足全部条件，默认：`false`
```

##### 获取条件详细说明：
条件由字符串构成，多个条件之间用 `,` 逗号隔开。
一个条件由： 条件键 + 条件符号 + 匹配值。
<br />
条件键：数据是对象或数组时有效，选填。
条件符号：包括 `<`, `>`, `<=`, `>=`, `!=`, `=` ，此部分如果不填则会使用 `=`。
